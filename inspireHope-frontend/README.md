# InspireHope Foundation Frontend

Modern React frontend for the InspireHope Foundation platform and landing page.

## Mission

Igniting change, uniting hearts, and uplifting communities.

## Vision

Envisioning a world where everyone belongs and no one is left out.

## Tech Stack

- React
- Vite
- CSS

## Current Status

Implemented so far:

- Responsive single-page landing experience
- Sticky navigation bar
- Mobile hamburger menu with slide-down navigation
- Improved mobile spacing and nav alignment
- Netlify deployment support with SPA redirects

## Run Locally

```bash
npm install
npm run dev
```

The app runs by default at `http://localhost:5173`.

## Build

```bash
npm run build
```

## Project Files

- `src/App.jsx` - homepage layout and content
- `src/App.css` - page-specific styles
- `src/index.css` - global theme and base styles
